# Local Development

### Make sure do you have
- `node` >= 16
- `npm` >= 7

## Installation

```
npm install
```

### First of all you must build and start core dev server
1. Go to packages/core
2. Build package

```shell
npm run build
```

3. Start dev server

```shell
npm run server
```

#### Watch mode
```shell
# Start dev server and build --watch
npm start
```

### After that build and start demo dev server
1. Go to packages/demo
2. Start dev server
```shell
npm run server
```

#### Watch mode
```shell
# Start dev server and build --watch
npm start
```
